Sketch 1: Scatterplot

Hypothesis: The longer the hours spent outside, the higher my daily reflection.

X-axis: Hours Spent Outside (0–10)
Y-axis: Daily Reflection (0–5)
Representation: Scatterplot — bubble color shows moodOutside (1–5).

Design Details:For this graph, a color palette was selected to communicate emotional states in a clear and intuitive manner. Warm tones such as red represent more intense or heightened emotions, whereas cool tones such as green indicate calmer feelings. This warm-to-cool gradient enables viewers to quickly interpret emotional variations across different days. Additionally, the use of varying circle sizes enhances readability, reduces visual complexity, and supports more effective data interpretation.

This visualization examines whether spending more time outdoors increases self-reflection will related with purchased condition.

Sketch 2: Parallel Coordination Chart

Hypothesis: When both hours outside and money spent increase,
daily reflection tends to decrease.

Data used: HoursOutside, MoneySpent, DailyReflection
Units: hours (0–10), money ($0–200), reflection (0–5)
Representation: 2D plot with color scale for reflection.

Design Details: A Line Graph was chosen to present multiple variables simultaneously. Although this approach may initially appear complex, the application of distinct colors explicitly identified in the legend allows users to differentiate variables with ease and observe changes over time. The color separation improves overall clarity and provides structured foundation for analyzing variations among datasets.

This plot explores how lifestyle spending may influence self-reflection despite more time spent outside.

Sketch 3: Calendar table data collection

Hypothesis: As the number of locations visited decreases, the number of purchased meals also decreases, while mood outside tends to increase.

Data used: LocationsVisited, MealsPurchased, MoodOutside
Representation: Creative visual no axis, uses colors or pattern to show relational change.

Design Details:This visualization is designed based on a daily recording format commonly used in personal planning tools, such as calendar based to do lists. Employing a structure that aligns with familiar daily routines facilitates more consistent data collection and enhances user engagement. This format also allows day to day changes to be tracked more clearly, resulting in a more intuitive and accessible analysis process.
